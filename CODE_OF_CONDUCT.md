# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant] as its
code of conduct. Please read it before contributing.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

To report behaviour that violates this code, contact the project maintainers
via GitHub Issues (for non-sensitive reports) or by opening a private security
advisory on the repository. Maintainers will review and respond.

## Enforcement

Maintainers are responsible for clarifying and enforcing the standards of the
Contributor Covenant and may take any action they deem appropriate in response
to violations, up to and including removing contributions and banning
contributors from the project.
